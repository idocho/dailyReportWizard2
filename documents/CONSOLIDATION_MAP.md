## 확실히 네이티브로 남는 것 (에이전트)
**키를 만지거나 카톡을 조작하는 전부:**
- AI 생성 실제(개인 키 + 문체 + API 호출)
- 카톡 발송·톡방 정리·history 기록·SmartWait
- 위에 필요한 1회 설정: 엔진·개인 키·전송 환경 (턴키 마법사)
→ 에이전트는 **1회 설정 후 트레이 무조작**. daily 조작 0.

## 개선 항목 (✨)
- 입력·검토를 **태블릿 반응형 + 키보드 네비**로 (PC 고정 탈피)
- **전송 진행을 웹 실시간** 표시 + 에이전트 "전송 중 만지지마" 오버레이
- 전송 속도 설정 → **SmartWait 자동**으로 대체(사용자 설정 제거)
- 일괄 공지 이미지 **자동 축소**(이미 적용)
- 두 앱(태블릿 입력 + PC 처리) → **한 웹앱**으로 (핵심 사용성 개선)

## 신규 강점 (웹/모바일만 가능)
철학: 고스란히 이식이 아니라 — 좋은 건 계승, **웹이 더 잘하는 건 새롭게 살린다.**
- ✅ **톤 조절 재생성** (착수·검증) — "따뜻/간결/구체" 원탭 재생성. 사실은 데이터 유지, 어조만 조정 + 이전 작성본 참고. agent_worker `TONE_DIRECTIVES`, 웹 톤버튼.
- (후보·미착수) 📷 카메라 사진 첨부 · 📴 오프라인 입력+동기화 · ⚡ 빠른입력/음성메모 · Analyzer 미리보기 · 푸시 알림

## 폐기 (🗑) — 통합으로 사라지는 부채
