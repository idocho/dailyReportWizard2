DailyReportWizard는 두 부분입니다. ① 웹 앱 — 태블릿/PC 브라우저에서 로그인 후 매일의 수업·성적을 입력하고 리포트를 검토·전송 요청합니다. ② 강사 에이전트(Windows) — 본인 PC에서 AI 특이사항 생성과 카카오톡 전송을 처리합니다. 먼저 1~3부로 웹을 익히고, 카톡 전송·AI 생성을 쓰려면 4부에서 에이전트를 설치하세요.

1부 · 웹 접속 & 로그인

1

앱 주소로 접속

Chrome/Edge(또는 Safari)에서 아래 주소로 접속합니다. 9인치급 이상 태블릿(가로) 또는 PC 권장.

앱 주소https://dailyreportwizard.web.app

포털이 뜨면 안정 버전(실사용) 또는 최신 개발본을 고릅니다.

2

로그인

로그인 화면에서 캠퍼스 선택 → 본인 이름 → 비밀번호를 입력합니다. 계정은 관리자가 발급합니다(별도 가입·설정 위저드 없음).

  • 최초 로그인 시 비밀번호 변경이 요구될 수 있습니다.
  • 로그인하면 담당 수업·학생·설정이 자동으로 연결됩니다 — DB 주소나 경로를 직접 입력할 필요가 없습니다.
🔑
비밀번호 분실·계정 문제는 관리자에게 문의하세요. 비활성 계정은 로그인되지 않습니다.

3부 · AI 문체 설정 (선택)

1

⚙️ 설정 › AI설정

문체(말투)를 고릅니다 — 내 말투 자동(전송 노트 학습) 또는 4개 프리셋(따뜻·상세 / 정돈·균형 / 정보·코칭 / 간결·요점). 각 문체의 설명·예시가 아래에 표시됩니다.

개별 지침(선택) — 매 생성마다 AI에 함께 전달되는 추가 프롬프트입니다(예: "끝인사로 다음 수업 날짜 안내"). 전 학생 메시지에 공통 적용됩니다.

ℹ️
AI 엔진·API 키는 본인 PC 에이전트에서 설정합니다(4부). 웹에는 문체·지침만 둡니다.

4부 · 강사 에이전트 설치 (AI 생성 · 카톡 전송)

AI 특이사항 생성과 카카오톡 발송은 본인 PC의 강사 에이전트가 처리합니다(개인 API 키는 이 PC를 떠나지 않음). 평소 트레이에 상주하며 자동으로 동작합니다.

💻
설치 조건. Windows 10/11 PC + 카카오톡 PC 버전 설치·로그인. 전송은 카톡 창을 자동 제어하므로 발송 중 PC를 다른 작업에 쓰지 마세요.
1

에이전트 다운로드

아래 버튼으로 내려받습니다(설치 과정 없는 단일 실행 파일).

⬇ DRW AI Agent 다운로드

⚠️
서명 안 된 프로그램이라 "Windows의 PC 보호" 창이 뜨면 추가 정보 → 실행. 백신 차단 시 허용/신뢰 처리(사내 자체 제작이라 안전). exe는 고정 폴더에 두고 쓰세요(설정·자동시작이 그 위치 기준).
2

최초 1회 설정

실행하면 설정 폼이 뜹니다. 아래를 입력하고 저장하고 시작:

  • 캠퍼스 — 드롭다운 선택.
  • 본인 이름 — 웹 로그인명과 정확히 동일해야 합니다(작업 큐 매칭 키).
  • AI 엔진 — Gemini(무료·추천) / Claude / GPT 중 선택.
  • 개인 API 키 — 5부에서 발급(엔진별 개인 키, 본인 PC에 암호화 저장).
  • 카톡 방 접두사 — 학부모방 이름 앞 공통 문구(예: "오직 ").
  • Windows 시작 시 자동 실행 — 체크 권장.
🔗
이름이 웹 로그인명과 다르면 생성·전송 작업이 큐에서 처리되지 않습니다. 반드시 일치시키세요.
3

상주 & 동작

  • 저장 후 에이전트는 트레이에 상주하며 웹에서 들어온 생성·전송 작업을 자동 처리합니다(평상시 실 발송 모드).
  • 창의 ×(닫기)·최소화는 트레이로 숨김 — 종료가 아닙니다. 완전 종료는 트레이 아이콘 우클릭 → 종료.
  • 자동 실행을 켜두면 PC 로그인 시 트레이에서 자동 가동됩니다 — 한 번 설정 후 신경 쓸 일이 없습니다.
🖱️
전송 중에는 마우스·키보드를 건드리지 마세요(카톡 창 자동 조작). 전송상태는 웹 📤 리포트·전송 우측 전송 모니터에서 실시간 확인됩니다.
ℹ️
웹에서 AI 생성·전송을 눌렀는데 에이전트가 감지되지 않으면 설치/실행 안내 + 다운로드 팝업이 뜹니다. 에이전트를 실행한 뒤 다시 시도하세요.

에이전트의 AI 생성을 쓰려면 외부 엔진의 개인 API Key가 필요합니다(회사가 키를 제공하지 않음 — 강사 개인 발급). 엔진을 고르면 발급 방법이 보입니다.

⭐
추천: Gemini. 무료·카드 불필요(하루 요청 한도만). 처음이면 Gemini로 시작하세요.

매일 흐름: ① 수업 입력(수업 중 태블릿) → ② 리포트·전송(AI 생성·검토·발송) → 필요 시 ③ 일괄 공지. 성적은 별도 탭. 입력은 즉시 클라우드 저장되고, 로그인하면 최신 데이터가 자동으로 로드됩니다.

① 매일 수업 입력

사이드바 ✏️ 수업 입력 탭. 상단에서 오늘 수업의 진도·과제를, 아래에서 학생별 평가를 기록합니다.

1

대상 반 선택

담당 수업이 여러 개면 상단 탭(또는 그룹 탭)에서 입력할 반·과목을 고릅니다. 화면 제목에 반 · 학년학기 · 과목과 오늘 날짜가 표시됩니다.

🔧
제목 우측의 날짜 변경 버튼은 테스트용으로 다른 날짜의 데이터를 입력/확인할 때 씁니다. 평소엔 건드리지 않아도 됩니다.
2

진도 입력

상단 진도 칸에서 대단원 → 소단원 순으로 선택합니다(커리큘럼이 연결돼 있으면 목록이 자동으로 뜸). 단원 목록이 없거나 직접 적고 싶으면 ✏️ 직접 입력 을 고르세요. 아래 미리보기로 최종 표기를 확인합니다.

3

과제 입력

과제 칸에서 형식을 고릅니다: p. 페이지 / # 번호 / 직접 입력. 페이지·번호는 시작~끝을 적으면 p.12~p.20 처럼 자동 정리됩니다.

💡
진도·과제는 반 전체 공통이라 한 번만 입력하면 됩니다. 입력 즉시 자동 저장됩니다.
4

학생별 평가 입력

아래 👥 학생별 입력에서 학생마다 카드 한 장씩 채웁니다. 각 줄의 의미:

항목입력 방식
과제수행 등급 1개 선택(택1) + 필요 시 자주 쓰는 문구(복수 선택).
컨디션상태 1개 선택(화살표 아이콘).
이해도이해 수준 1개 + 세부 태그(복수) + 하이라이트.
참여·풀이수업 참여·풀이 태그(복수 선택).
주의주의 관찰 태그(복수 선택) — 졸음·잡담·계산실수 등.
시험시험 본 날만 — 시험 결과(복수 선택): 🏆우수·👍양호·⚠️실수실점·🧩심화아쉬움·📉아쉬움. AI가 결과에 맞춰 칭찬/격려를 자동 조정.
메모자유 서술 특이사항(한 줄).

모든 입력은 누를 때마다 즉시 저장됩니다. 별도 저장 버튼이 없습니다.

5

완료 표시(점) 읽기

학생 이름 옆 점과 상단 n/n명 완료 카운터로 진행 상황을 봅니다.

과제 평가 + 진도/과제 입력 완료
과제 평가는 했으나 진도/과제 미입력
아직 과제 평가 안 함

모든 학생이 초록색이 되도록 채우는 것이 목표입니다.

② 리포트 · 전송

사이드바 📤 리포트·전송 탭. 좌측 디렉토리 트리(반→학생, 오늘 수업 펼침)에서 학생을 고르면 가운데에 편집, 우측에 미리보기·전송 모니터가 뜹니다. 문체는 ⚙️ 설정 › AI설정에서 정합니다.

1

학생 선택 & AI 생성

트리에서 학생 클릭 → 가운데에 그 학생의 진도·과제·수행도·관찰 태그·강사 메모 요약이 보입니다. ✨ 생성을 누르면 에이전트가 데이터·메모·문체로 발송문을 만들어 옵니다. ✨ 일괄 생성은 활성 반 전체를 한 번에.

💡
발송문은 자유롭게 검토·수정하세요. 생성된 학생은 트리에 검토중, 아직이면 미생성 뱃지로 표시됩니다.
2

발송 제외 & 미리보기

상단 발송 진도/과제 칩으로 특정 교재를 이번 발송에서 ✕ 제외할 수 있습니다(메시지·미리보기·AI 생성에서 함께 빠짐). 우측에 실제 카톡 발송 형식 미리보기가 표시됩니다.

3

전송 — 여러 반 한 번에

전송 → 누르면 검토중(발송문 작성) 학생이 여러 반에 걸쳐 표시됩니다. 반 헤더/학생 체크로 대상을 고르고 전송 시작 → 반별로 큐에 쌓여 순차 발송됩니다. 미생성·미검토 학생은 자동 제외.

📡
우측 전송 모니터(다크 패널)에서 작업별 진행·완료·실패를 실시간 확인하고, 진행 중 작업을 취소하거나 완료 건을 정리할 수 있습니다.
🤖
생성·전송은 본인 PC 에이전트가 처리합니다. 에이전트가 꺼져 있으면 안내 팝업이 뜨니 실행 후 다시 시도하세요.

③ 일괄 공지

사이드바 📢 일괄 공지 탭. 데일리 리포트와 같은 화면 구성 — 좌측 트리에서 여러 반 학생을 체크해 같은 메시지를 일괄 발송합니다(데일리 리포트와 별개).

1

수신자 선택 & 작성

트리 체크박스로 반 단위/학생 단위로, 여러 반을 가로질러 선택합니다. 가운데에서 메시지 작성 — 저장 템플릿 불러오기, 변수 {이름} {반} {날짜} 삽입, 이미지 첨부 가능. 우측 미리보기로 확인.

2

일괄 전송

📢 일괄 전송 → → 선택한 전 학생에게 발송(변수는 학생별로 치환). 전송 모니터에서 진행 확인.

④ 성적 입력

사이드바 📊 성적 입력 탭. 좌측에서 반·과목을 고른 뒤 시험을 추가하고 원점수를 입력합니다.

1

새 시험 추가

+ 새 시험 추가 클릭 → 시험 유형, 회차(선택), 날짜, 만점, 메모(시험 범위 등)를 설정합니다.

2

점수 입력 & 저장

학생별 점수 칸에 원점수를 입력합니다. 평균 등 통계가 자동 계산됩니다. 입력 후 저장 하세요.

⚠️
입력 권한. 주간/단원 시험은 해당 과목 담당 강사만, 학년단위 시험은 담임만 입력할 수 있습니다. 권한이 없으면 저장 시 안내 토스트가 뜹니다.

⚙️ 설정 상세

항목용도
🤖 AI설정문체(말투) 선택 + 개별 지침(추가 프롬프트). AI 발송문 톤을 정함.

문제 해결 (FAQ)

로그인이 안 돼요

캠퍼스·이름·비밀번호를 확인하세요. 이름은 관리자가 발급한 표기와 정확히 같아야 합니다. "비활성화된 계정" 안내가 뜨면 관리자에게 문의하세요.

AI 생성/전송을 눌렀는데 안 돼요

본인 PC 강사 에이전트가 실행 중이어야 합니다. 안내 팝업이 뜨면 에이전트를 실행(트레이 확인)한 뒤 다시 시도하세요. 에이전트의 이름이 웹 로그인명과 동일한지도 확인하세요.

AI 생성에서 한도·503 오류가 떠요

엔진 일시 과부하/일일 한도입니다. 잠시 후 재시도하면 보통 해결됩니다(에이전트가 일시 오류는 자동 재시도). Gemini에서 limit:0이면 새 프로젝트로 키를 재발급하세요.

성적 저장이 안 돼요

입력 권한 문제일 수 있습니다. 주간/단원 시험은 과목 담당, 학년단위 시험은 담임만 저장 가능합니다. 역할을 📚 내 담당 수업에서 확인하세요.

전송 중인데 엉뚱한 방에 갈까 걱정돼요

에이전트는 채팅방이 실제로 열렸는지 확인한 뒤에만 본문을 보냅니다(엉뚱한 방·반복 전송 차단). 다만 전송 중에는 마우스·키보드를 건드리지 마세요. 실패 건은 모니터에 표시됩니다.
